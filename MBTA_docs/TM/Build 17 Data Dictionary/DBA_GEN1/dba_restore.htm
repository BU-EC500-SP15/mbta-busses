TransitMaster SQL Server Database Restore

Any database being restored, whether daily logs or the full summary database, must be restored to the ORIGINAL size with exactly the same database name. All original sizing will be configured specifically for each property, and documented accordingly.

    1. Select Database Backup/Restore from the Tools menu within SQL Enterprise Manager.

    2. Select the Restore tab within the newly arrived window.

    3. Select From Device
    4. Choose the Tape device storing the desired database
    5. Choose the database you want to restore off of the tape. Please note that the database which has been selected will have the diamond shape filled in with blue when it has been properly chosen.
    6. Click on the Down arrow in the Destination Databases to reveal a list of databases to restore into.
    7. Choose "New" from this list of databases to restore TO

    8.  
    9. A new window will appear asking you to input data concerning the database to be created.

    10. Enter the database name
    11. Enter the device to which data will be restored (specified in property documentation)
    12. Enter the device to which log data will be restored (specified in property documentation)
    13. Enter the amount of space to be utilized within the data device (This is specified for each database based upon property. This size MUST be the same as the original size when the database was backed up.)
    14. Enter the amount of space to be utilized within the log device (This is specified for each database based upon property. This size MUST be the same as the original size when the database was backed up.)
    15. Select Create Database
    16. After the database has been created by the system, select Restore Now from the restore tab.
    17. The database will now be restored to its original location and can be utilized by the system.

If any errors occur during either backup, or restoration, please contact the network administrator for your system. If immediate assistance is needed, TransitMaster personnel are on staff to aid with any problems that arise.
