Recommended Tape Replacement Interval

The following list describes the recommended interval at which each property must utilize new tapes in the Daily Log Database backup process. This is due to the fact that each daily log contains enough information that one tape will not be adequate to store a year’s worth of data.

  50 buses 100 buses 150 buses 200 buses
New Tape Interval: 6 months per tape SET 4 months per tape SET 2 months per tape SET 2 months per tape SET

Even though the tape may not be completely full, it is recommended that these intervals be followed in order to maintain a satisfactory level of organization in the backup process. In this manner, a tape should not become full in the middle of a month.
